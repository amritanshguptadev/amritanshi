# Amritanshi – Frontend Developer Portfolio

A personal portfolio website built with HTML5, CSS3, and Vanilla JavaScript.

## 🌐 Live Demo
Deployed on Vercel: [amritanshi.vercel.app](https://amritanshi.vercel.app)

## ✨ Features
- Responsive dark-mode design with glassmorphism
- Animated hero section with typewriter effect
- Scroll-reveal animations
- Animated skill progress bars
- Interactive contact form
- Mobile-friendly navigation
- SEO optimised

## 🛠️ Tech Stack
- HTML5
- CSS3 (Vanilla — no frameworks)
- JavaScript (Vanilla)

## 📁 Project Structure
```
amritanshiportfolio/
├── index.html      # Main HTML file
├── style.css       # Styles
├── script.js       # JavaScript
├── profile.png     # Profile photo
├── vercel.json     # Vercel deployment config
└── README.md
```

## 🚀 Deploy on Vercel
1. Push to GitHub
2. Import repo at [vercel.com/new](https://vercel.com/new)
3. No build step needed – it auto-detects static HTML
